# cf worker 保 serv 账号，不保活
按照serv的封禁趋势，禁止搭建代理是迟早的事，且serv本身正确的用途是建站而不是代理，因此我重置了自己所有的serv服务器，用来搭建各种服务和数据库

但是serv有登录要求，90天不登录可能会被封号，因此诞生了这个项目，参考了天诚的保活项目，也就是文件夹内的_worker.js
